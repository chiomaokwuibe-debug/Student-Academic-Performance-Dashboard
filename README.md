# Student Academic Performance Dashboard

## Project Overview
This Power BI dashboard analyzes student academic performance, attendance, study hours, and test scores. It provides insights into factors that influence student achievement through interactive visualizations.

## Tools Used
- Power BI
- Data Modeling
- Data Visualization

## Key Features
- Student performance analysis
- Average score tracking
- Attendance monitoring
- Study hours analysis
- Gender-based student distribution
- Interactive filters and slicers

## Insights
- Identifies trends in student academic performance
- Examines the relationship between study hours and test scores
- Compares performance across different courses
- Highlights attendance patterns and their impact on results
